# 개인정보처리방침 / Privacy Policy

**Parfum AI** ("앱")는 사용자의 개인정보를 소중히 여깁니다.

최종 수정일: 2026년 5월 31일

## 수집하는 정보
본 앱은 다음 정보만 처리합니다:
- MBTI, 나이대, 패션 스타일, 향수 취향 등 **사용자가 직접 입력한 취향 데이터**
- 해당 데이터는 향수 추천 생성에만 사용되며 서버에 저장되지 않습니다

## 수집하지 않는 정보
- 이름, 이메일, 전화번호 등 개인 식별 정보
- 위치 정보
- 기기 고유 식별자

## 제3자 서비스
본 앱은 AI 추천 생성을 위해 **Anthropic Claude API**를 사용합니다.
- Anthropic 개인정보처리방침: https://www.anthropic.com/privacy

# Privacy Policy

**Parfum AI** does not collect or store any personal information.

The taste preference data (MBTI, age, fashion style, scent preferences) entered by users is used solely to generate perfume recommendations and is not stored on our servers.
